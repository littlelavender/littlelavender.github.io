# littlelavender.github.io
一个以[Astro](https://docs.astro.build)构建的个人博客，使用[🌸Sakura](https://github.com/LIlGG/halo-theme-sakura)主题。

## 运行

以开发模式启动。

在开发模式启动时搜素功能不可用。

```sh 
npm install
npm run dev
```

打包和预览。

```sh
npm run build
npx pagefind --site dist/posts --output-path dist/pagefind
npm run preview
```
